# Glimzy
Real-time filters to faces using web camera input

Glimzy Photobooth is a web-based photobooth application that allows users to capture a series of photos using their webcam and apply fun face filters in real time. The application automatically generates a downloadable photo strip after taking four consecutive photos.

## ✨ Features

- 🎥 Live webcam preview
- 😊 Real-time face detection using Face API
- 🎭 Multiple face filters
  - Bunny
  - Hat
  - Sunglasses
  - Flower Crown
  - Cat Ears
  - Dog Ears
  - Pink Ears
- ⏱️ Automatic 3-second countdown
- 📷 Capture 4 photos automatically
- 🖼️ Photo strip generation
- 💾 Download photo strip as PNG
- 🔄 Retake photos anytime
